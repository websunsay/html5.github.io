HTML елемент <q> для коротких цитат

Елемент HTML <q> визначає коротку цитату. Браузери зазвичай вставляють лапки навколо елемента <q>.

Приклад:

<p>Мета WWF - це: <q>Створити майбутнє, де люди житимуть у гармонії з природою. </q></p>
Спробуйте самі! »

HTML елемент <blockquote> для цитування

HTML елемент <blockquote> визначає розділ, який цитується з іншого джерела. Браузери зазвичай відображають елементи <blockquote> з відступом.

Приклад:

<p>Here is a quote from WWF's website:</p>
<blockquote cite="http://www.worldwildlife.org/who/index.html">
For 50 years, WWF has been protecting the future of nature.
The world's leading conservation organization,
WWF works in 100 countries and is supported by
1.2 million members in the United States and
close to 5 million globally.
</blockquote>
Спробуйте самі! »

HTML елемент<abbr> для абревіатур

Елемент HTML <abbr> визначає абревіатуру або акронім. Маркування абревіатур може дати корисну інформацію для браузерів, систем перекладу та пошукових систем.

Приклад:

<p>The <abbr title="World Health Organization">WHO</abbr> was founded in 1948.</p>
Спробуйте самі! »

HTML елемент <address> для контактної інформації

HTML елемент <address> визначає контактну інформацію (автор / власник) документа або статті. Елемент <address> за умовчанням відображається курсивом. Більшість браузерів додають розрив рядків до і після елемента.

Приклад:

<address>
Written by John Doe.<br>
Visit us at:<br>
Example.com<br>
Box 564, Disneyland<br>
USA
</address>

HTML елемент <cite> для назви роботи

HTML елемент <cite> визначає назву твору. Браузери зазвичай показують <cite> елементи курсивом.

Приклад:

<p><cite>The Scream</cite> by Edvard Munch. Painted in 1893.</p>
Спробуйте самі! »

HTML елемент <bdo> для двонаправленого перевизначення

HTML елемент <bdo> використовується для перевизначення поточного напрямку тексту.

Приклад:

<bdo dir="rtl">This text will be written from right to left</bdo>
