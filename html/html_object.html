Для чого потрібні плагіни?

Метою плагіна є розширення функціональності веб-браузера.

HTML Помічники (плагіни)

Допоміжні програми (плагіни) - це комп'ютерні програми, які розширюють стандартну функціональність веб-браузера.

Прикладами відомих плагінів є Java-аплети.

Плагіни можна додавати до веб-сторінок за допомогою тегу <object> або тегу <embed>.

Плагіни можна використовувати для багатьох цілей: відображення карт, пошук вірусів, перевірка вашого банківського ідентифікатора тощо.

Щоб відобразити відео та аудіо: використовуйте <video> і <audio> теги.

Елемент <object> (об'єкт)

Елемент <object> підтримується всіма браузерами.

Елемент <object> визначає вбудований об'єкт у документі HTML.

Він використовується для вбудовування плагінів (наприклад, Java-аплетів, PDF-рідерів, Flash-плеєрів) у веб-сторінки.

Приклад:

<object width="400" height="50" data="bookmark.swf"></object>

Елемент <embed>

Елемент <embed> підтримується у всіх основних браузерах.

Елемент <embed> також визначає вбудований об'єкт в HTML документі.

Веб-браузери підтримували елемент <embed> довгий час. Проте він не був частиною специфікації HTML до специфікації HTML5.

Приклад:

<embed width="400" height="50" src="bookmark.swf">

Зверніть увагу, що елемент <embed> не має закриваючого тега. Він не може містити альтернативного тексту.

Елемент <embed> також можна використовувати для включення HTML в HTML:

Приклад:

<embed width="100%" height="500px" src="snippet.html">

Приклад:
