Ваш браузер не підтримує елемент <canvas>.

HTML елемент <canvas> (с анг. canvas - полотно) використовується для малювання графіки на веб-сторінці.

Графічний об'єкт ліворуч створюється за допомогою <canvas>. Він показує чотири елементи: червоний прямокутник, градієнтний прямокутник, багатобарвний прямокутник і багатобарвний текст.

Що таке HTML Canvas?

HTML елемент <canvas> використовується для малювання графіки на льоту за допомогою JavaScript.

Елемент <canvas> є лише контейнером для графіки. Ви повинні використовувати JavaScript, щоб насправді малювати графіку.

Canvas має декілька методів для малювання доріжок, полів, кіл, тексту та додавання зображень.

Canvas Приклади

Canvas (полотно) - це прямокутна область на HTML сторінці. За замовчуванням полотно не має кордону та вмісту.

Розмітка виглядає так:

<canvas id="myCanvas" width="200" height="100"></canvas>

Примітка: Завжди вказуйте атрибут id (на який посилається скрипт), атрибути width (ширина) та height (висота) для визначення розміру полотна. Щоб додати кордон, скористайтеся атрибутом style.
