HTML Аудіо і Відео DOM Довідник

HTML5 DOM має методи, властивості та події для <audio> та <video> елементів.

Ці методи, властивості і події дозволяють вам маніпулювати <audio> та <video> елементами використовуючи JavaScript.

HTML Аудіо/Відео Методи

Метод Опис
addTextTrack() Додає новий текстовий трек до аудіо/відео
canPlayType() Перевіряє, чи браузер може відтворювати вказаний тип аудіо/відео
load() Перезавантажує аудіо/відео елемент
play() Починає відтворення аудіо/відео
pause() Призупиняє поточне відтворення аудіо/відео

HTML Аудіо/Відео Властивості

Властивість Опис
audioTracks Повертає об'єкт AudioTrackList, що представляє доступні звукові доріжки
autoplay Встановлює або повертає, чи слід аудіо/відео розпочати відтворення, як тільки воно буде завантажено
buffered Повертає об'єкт TimeRanges, який представляє буферизовані частини аудіо/відео
controller Повертає об'єкт MediaController, що представляє поточний контролер медіа аудіо/відео
controls Встановлює або повертає, чи аудіо/відео має відображати елементи керування (наприклад, відтворення/пауза тощо.)
crossOrigin Встановлює або повертає настройки CORS аудіо/відео
currentSrc Повертає URL поточного аудіо/відео
currentTime Встановлює або повертає поточну позицію відтворення в аудіо/відео (у секундах)
defaultMuted Встановлює чи повертає аудіо/відео за замовчуванням
defaultPlaybackRate Встановлює або повертає стандартну швидкість відтворення аудіо/відео
duration Повертає тривалість поточного аудіо / відео (у секундах)
ended Повертає, чи закінчилося відтворення аудіо/відео
error Повертає об'єкт MediaError, що представляє стан помилки аудіо/відео
loop Встановлює або повертає, чи аудіо/відео слід починати знову після завершення
mediaGroup Встановлює або повертає групу, до якої належить аудіо/відео (використовується для з'єднання декількох аудіо/відео елементів)
muted Встановлює або повертає, чи вимкнено аудіо/відео, чи ні
networkState Повертає поточний стан мережі аудіо/відео
paused Повертає якщо аудіо/відео призупинено або ні
playbackRate Встановлює або повертає швидкість відтворення аудіо/відео
played Повертає об'єкт TimeRanges, що представляє відтворювані частини аудіо / відео
preload Встановлює або повертає, чи потрібно завантажувати аудіо/відео під час завантаження сторінки
readyState Повертає поточний стан готовності аудіо/відео
seekable Повертає об'єкт TimeRanges, який представляє доступні для пошуку частини аудіо/відео
seeking Повертає, чи шукає користувач в даний момент аудіо/відео
src Встановлює або повертає поточне джерело аудіо/відео елемента
startDate Повертає об'єкт Date, що представляє поточний зсув часу
textTracks Повертає об'єкт TextTrackList, що представляє доступні текстові доріжки
videoTracks Повертає об'єкт VideoTrackList, що представляє доступні відеотреки
volume Встановлює або повертає гучність аудіо/відео

HTML Аудіо/Відео Події

Подія Опис
abort Спрацьовує, коли завантаження аудіо/відео перервано
canplay Спрацьовує, коли браузер може запустити відтворення аудіо/відео
canplaythrough Спрацьовує, коли браузер може відтворювати аудіо/відео без зупинки для буферизації
durationchange Спрацьовує, коли змінюється тривалість аудіо/відео
emptied Спрацьовує, коли поточний список відтворення порожній
ended Спрацьовує після завершення поточного списку відтворення
error Спрацьовує, коли сталася помилка під час завантаження аудіо/відео
loadeddata Спрацьовує, коли браузер завантажує поточний кадр аудіо/відео
loadedmetadata Спрацьовує, коли веб-браузер завантажив метадані для аудіо/відео
loadstart Спрацьовує, коли браузер починає шукати аудіо/відео
pause Спрацьовує, коли аудіо/відео було призупинено
play Спрацьовує, коли аудіо/відео було запущено або більше не призупинено
playing Спрацьовує, коли аудіо/відео відтворюється після призупинення або зупинки для буферизації
progress Спрацьовує, коли браузер завантажує аудіо/відео
ratechange Спрацьовує при зміні швидкості відтворення аудіо/відео
seeked Спрацьовує, коли користувач закінчує переміщення/пропуск до нової позиції в аудіо/відео
seeking Спрацьовує, коли користувач починає переміщення/пропуск до нової позиції в аудіо/відео
stalled Спрацьовує, коли веб-браузер намагається отримати медіа-дані, але дані недоступні
suspend Спрацьовує, коли веб-браузер навмисно не отримує медіаданих
timeupdate Спрацьовує, коли змінюється поточна позиція відтворення
volumechange Спрацьовує, при зміні гучності
waiting Спрацьовує, коли відео зупиняється, оскільки потрібно буферизувати наступний кадр
