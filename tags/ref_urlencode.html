Кодування URL перетворює символи у формат, який може бути переданий через Інтернет.

URL (Uniform Resource Locator) - Уніфікований покажчик інформаційного ресурсу

Веб-браузери запитують сторінки з веб-серверів за допомогою URL-адреси.

URL-адреса - це адреса веб-сторінки, наприклад: https://www.w3schools.com.

Кодування URL (кодування у відсотках)

URL-адреси можна надсилати лише через Інтернет за допомогою ASCII набору символів.

Оскільки URL-адреси часто містять символи поза набором ASCII, URL-адреса має бути перетворена у правильний формат ASCII.

Кодування URL замінює небезпечні символи ASCII на "%", за якими слідують дві шістнадцяткові цифри.

URL-адреси не можуть містити пробіли. Кодування URL зазвичай замінює пробіл знаком плюс (+) або %20.

URL Encoding Functions

In JavaScript, PHP, and ASP there are functions that can be used to URL encode a string.

PHP has the rawurlencode() function, and ASP has the Server.URLEncode() function.

In JavaScript you can use the encodeURIComponent() function.

Click the "URL Encode" button to see how the JavaScript function encodes the text.

Note: The JavaScript function encodes space as %20.
