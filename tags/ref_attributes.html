Всі HTML Атрибути

В даній таблиці зібрані всі атрибути, які використовуються в HTML5 на даний момент.

Атрибут Належить до: Опис
accept <input> Визначає типи файлів, які приймає сервер (тільки для type="file")
accept-charset <form> Визначає кодування символів, які будуть використовуватися для подання форми
accesskey Глобальні атрибути Визначає клавішу швидкого доступу для активації/фокусування елемента
action <form> Вказує, куди надсилати дані форми після відправки форми
align Не підтримується в HTML 5. Визначає вирівнювання відповідно до оточуючих елементів. Замість цього використовуйте CSS
alt <area>, <img>, <input> Визначає альтернативний текст, коли початковий елемент не відображається
async <script> Вказує, що сценарій виконується асинхронно (лише для зовнішніх скриптів)
autocomplete <form>, <input> Вказує, чи повинен елемент <form> або <input> мати автозавершення
autofocus <button>, <input>, <select>, <textarea> Вказує, що елемент повинен автоматично отримувати фокус при завантаженні сторінки
autoplay <audio>, <video> Визначає, що аудіо/відео файл почне відтворюватись, як тільки буде готовий
bgcolor Не підтримується в HTML 5. Визначає колір фону елемента. Замість цього використовуйте CSS
border Не підтримується в HTML 5. Визначає ширину (товщину) кордону елемента. Замість цього використовуйте CSS
charset <meta>, <script> Визначає кодування символів
checked <input> Вказує, що елемент <input> повинен бути попередньо вибраний при завантаженні сторінки (для type="checkbox" або type="radio")
cite <blockquote>, <del>, <ins>, <q> Вказує URL адресу, яка пояснює цитований/видалений/вставлений текст
class Глобальні атрибути Визначає одне або кілька імен класів для елемента (відноситься до класу в таблиці стилів)
color Не підтримується в HTML 5. Визначає колір тексту елемента. Замість цього використовуйте CSS
cols <textarea> Визначає видиму ширину текстової області
colspan <td>, <th> Вказує кількість стовпців, яким повинна поширюватися клітинка таблиці
content <meta> Дає значення, пов'язане з атрибутом http-equiv або name
contenteditable Глобальні атрибути Вказує, чи можна редагувати вміст елемента
controls <audio>, <video> Вказує, що елементи керування аудіо/відео повинні відображатися (наприклад, кнопка відтворення/паузи тощо)
coords <area> Визначає координати області
data <object> Вказує URL ресурсу, який буде використовуватися об'єктом
data-* Глобальні атрибути Використовується для зберігання приватних даних на сторінці або програмі
datetime <del>, <ins>, <time> Визначає дату та час
default <track> Вказує, що трек має бути включений, якщо налаштування користувача не вказують на те, що інший трек буде більш доцільним
defer <script> Вказує, що сценарій виконується після завершення аналізу сторінки (лише для зовнішніх скриптів)
dir Глобальні атрибути Вказує напрямок тексту для вмісту елемента
dirname <input>, <textarea> Вказує, що напрямок тексту буде надіслано
disabled <button>, <fieldset>, <input>, <optgroup>, <option>, <select>, <textarea> Вказує, що вказаний елемент / група елементів має бути вимкнено
download <a>, <area> Вказує, що ціль буде завантажена, коли користувач натискає на гіперпосилання
draggable Глобальні атрибути Вказує, чи можна перетягувати елемент
dropzone Глобальні атрибути Визначає, чи будуть перетягувані дані копіюватись, переміщуватись або зв'язуватись, коли вони будуть скинуті
enctype <form> Визначає, як кодувати дані форми при передачі на сервер (тільки для method="post")
for <label>, <output> Вказує, до якого елементу(ам) прив'язана мітка/обчислення
form <button>, <fieldset>, <input>, <label>, <meter>, <object>, <output>, <select>, <textarea> Вказує назву форми, до якої належить елемент
formaction <button>, <input> Вказує, куди надсилати дані форми після подання форми. Тільки для type="submit"
headers <td>, <th> Вказує одну або кілька клітин заголовків, з якими пов'язана клітина
height <canvas>, <embed>, <iframe>, <img>, <input>, <object>, <video> Визначає висоту елемента
hidden Глобальні атрибути Вказує, що елемент ще не є або більше не є відповідним (релевантним)
high <meter> Вказує діапазон, який вважається високим значенням
href <a>, <area>, <base>, <link> Вказує URL-адресу сторінки, на яку переходить посилання
hreflang <a>, <area>, <link> Визначає мову пов'язаного документа
http-equiv <meta> Надає HTTP-заголовок для інформації/значення атрибута вмісту
id Глобальні атрибути Вказує унікальний ідентифікатор елемента
ismap <img> Вказує зображення як карту зображення на стороні сервера
kind <track> Визначає тип текстового треку
label <track>, <option>, <optgroup> Вказує назву текстового треку
lang Глобальні атрибути Вказує мову вмісту елемента
list <input> Відноситься до елементу <datalist> який містить заздалегідь визначені параметри для елементу <input>
loop <audio>, <video> Вказує, що аудіо / відео почнеться знову, кожен раз після завершення
low <meter> Вказує діапазон, який вважається низьким значенням
max <input>, <meter>, <progress> Вказує максимальне значення
maxlength <input>, <textarea> Вказує максимальну кількість символів, дозволених у елементі
media <a>, <area>, <link>, <source>, <style> Вказує, для якого медіа/пристрою оптимізований пов'язаний документ
method <form> Вказує метод HTTP, який слід використовувати при надсиланні даних форм
min <input>, <meter> Вказує мінімальне значення
multiple <input>, <select> Вказує, що користувач може вводити більше одного значення
muted <video>, <audio> Визначає, що аудіовиход відео повинен бути приглушений (вимкнений)
name <button>, <fieldset>, <form>, <iframe>, <input>, <map>, <meta>, <object>, <output>, <param>, <select>, <textarea> Вказує назву елемента
novalidate <form> Вказує, що форма не повинна перевірятися під час відправки
onabort <audio>, <embed>, <img>, <object>, <video> Скрипт, який буде виконуватися під час скасування
onafterprint <body> Скрипт, який буде запущено після друку документа
onbeforeprint <body> Скрипт, що запускається перед друком документа
onbeforeunload <body> Скрипт, який буде запущено, коли документ буде вивантажено
onblur Всі видимі елементи. Скрипт буде запущено, коли елемент втрачає фокус
oncanplay <audio>, <embed>, <object>, <video> Скрипт, який буде запущено, коли файл буде готовий до початку відтворення (коли він достатньо буферизований для початку)
oncanplaythrough <audio>, <video> Скрипт, який потрібно запустити, коли файл можна відтворити до кінця без призупинення для буферизації
onchange Всі видимі елементи. Скрипт, який буде виконуватися, коли змінюється значення елемента
onclick Всі видимі елементи. Скрипт, який буде запущено під час натискання елемента
oncontextmenu Всі видимі елементи. Скрипт, який потрібно запустити, коли спрацьовує контекстне меню
oncopy Всі видимі елементи. Скрипт, який буде виконуватися при копіюванні вмісту елемента
oncuechange <track> Скрипт, який буде виконуватися при зміні мітки в елементі <track>
oncut Всі видимі елементи. Скрипт, який буде виконуватися, коли змінюється вміст елемента
ondblclick Всі видимі елементи. Скрипт, який буде запущено при подвійному натисканні на елементі
ondrag Всі видимі елементи. Скрипт, який буде запущено, коли елемент перетягується
ondragend Всі видимі елементи. Скрипт буде запущено в кінці операції перетягування
ondragenter Всі видимі елементи. Скрипт, який буде запущено, коли елемент перетягується до дійсного цільового скидання
ondragleave Всі видимі елементи. Скрипт, який буде запущено, коли елемент залишає дійсну мету скидання
ondragover Всі видимі елементи. Скрипт, який буде запущено, коли елемент перетягується через дійсну мету скидання
ondragstart Всі видимі елементи. Скрипт, який буде запущено на початку операції перетягування
ondrop Всі видимі елементи. Скрипт, який буде запущено, коли елемент перетягується
ondurationchange <audio>, <video> Скрипт буде запущено, коли змінюється тривалість медіа
onemptied <audio>, <video> Скрипт буде запускатися, коли трапиться щось погане та файл раптово стане недоступним (наприклад, несподівано відключиться)
onended <audio>, <video> Скрипт, який буде запущено, коли медіа досягнуть кінця (корисна подія для повідомлень типу "спасибі за прослуховування")
onerror <audio>, <body>, <embed>, <img>, <object>, <script>, <style>, <video> Сценарій, який буде запускатися при виникненні помилки
onfocus Всі видимі елементи. Скрипт, який потрібно запустити, коли елемент отримує фокус
onhashchange <body> Скрипт, який потрібно запустити, коли відбулася зміна прив'язки частини URL-адреси
oninput Всі видимі елементи. Скрипт, який потрібно запустити, коли елемент отримує користувацький ввод
oninvalid Всі видимі елементи. Скрипт, який буде запущено, коли елемент недійсний
onkeydown Всі видимі елементи. Скрипт буде запущено, коли користувач натискає клавішу на клавіатурі
onkeypress Всі видимі елементи. Скрипт буде запущено, коли користувач натискає клавішу на клавіатурі
onkeyup Всі видимі елементи. Скрипт буде запущено, коли користувач відпускає клавішу
onload <body>, <iframe>, <img>, <input>, <link>, <script>, <style> Скрипт буде запущено, коли буде завершено завантаження елементу
onloadeddata <audio>, <video> Скрипт буде запущено при завантаженні медіаданих
onloadedmetadata <audio>, <video> Скрипт буде запущено, коли завантажуються метадані (наприклад, розміри та тривалість)
onloadstart <audio>, <video> Скрипт буде запущено так само, як файл починає завантажуватися, перш ніж все буде завантажено
onmousedown Всі видимі елементи. Скрипт буде запущено, коли кнопка миші натиснута на елемент
onmousemove Всі видимі елементи. Скрипт буде запущено, поки вказівник миші переміщується над елементом
onmouseout Всі видимі елементи. Скрипт буде запущено, коли вказівник миші переміститься з елемента
onmouseover Всі видимі елементи. Скрипт буде запущено, коли вказівник миші переміщується над елементом
onmouseup Всі видимі елементи. Скрипт буде запущено, коли кнопка миші буде відпущена над елементом
onmousewheel Всі видимі елементи. Скрипт буде запущено, коли колесо миші прокручується над елементом
onoffline <body> Скрипт буде запущено, коли браузер почне працювати в автономному режимі
ononline <body> Скрипт буде запущено, коли браузер почне працювати в Інтернеті
onpagehide <body> Скрипт буде запущено, коли користувач переміщується зі сторінки
onpageshow <body> Скрипт буде запущено, коли користувач переходить на сторінку
onpaste Всі видимі елементи. Скрипт буде запущено, коли користувач вставить певний вміст у елемент
onpause <audio>, <video> Скрипт буде запущено, коли носій буде призупинено користувачем або програмно
onplay <audio>, <video> Скрипт буде запущено під час відтворення носія
onplaying <audio>, <video> Скрипт буде запущено під час відтворення носія
onpopstate <body> Скрипт буде запущено, коли історія вікна зміниться.
onprogress <audio>, <video> Скрипт буде запущено, коли браузер перебуває в процесі отримання медіа-даних
onratechange <audio>, <video> Скрипт буде запускатися кожного разу, коли змінюється швидкість відтворення (наприклад, коли користувач переходить у режим повільного перегляду або перемотування вперед).
onreset <form> Скрипт буде запущено після натискання кнопки скидання у формі.
onresize <body> Скрипт буде запущено під час зміни розміру вікна веб-браузера.
onscroll Всі видимі елементи. Скрипт буде запущено, коли прокручується смуга прокрутки елемента
onsearch <input> Сценарій буде запущено, коли користувач запише щось у полі пошуку (для <input="search">)
onseeked <audio>, <video> Скрипт буде запущено, коли атрибут шукає значення false, що вказує на те, що пошук закінчився
onseeking <audio>, <video> Скрипт буде запущено, коли атрибут шукає значення true, що вказує на те, що пошук є активним
onselect Всі видимі елементи. Скрипт буде запущено, коли елемент буде вибрано
onstalled <audio>, <video> Скрипт буде запущено, коли веб-браузер не може отримати дані медіа з будь-якої причини
onstorage <body> Скрипт буде запущено, коли оновлено область зберігання Web
onsubmit <form> Скрипт буде запущено після відправки форми
onsuspend <audio>, <video> Скрипт, запущений під час завантаження медіаданих, зупиняється, перш ніж він буде повністю завантажений з будь-якої причини
ontimeupdate <audio>, <video> Скрипт буде запущено, коли змінено позицію відтворення (наприклад, коли користувач швидко переходить в іншу точку в медіа)
ontoggle <details> Скрипт буде запущено, коли користувач відкриє або закриє елемент <details>
onunload <body> Скрипт буде запущено, коли сторінка вивантажена (або закрито вікно веб-браузера)
onvolumechange <audio>, <video> Скрипт, який буде запускатися при кожній зміні гучності відео/аудіо
onwaiting <audio>, <video> Скрипт буде запущено, коли носій призупинено, але очікується, що його буде відновлено (наприклад, коли носій призупиняється для буферизації більшої кількості даних)
onwheel Всі видимі елементи. Скрипт буде запущено, коли колесо миші прокручується вниз або вгору над елементом
open <details> Вказує, що дані повинні бути видимими (відкритими) для користувача
optimum <meter> Вказує, яке значення є оптимальним значенням для датчика
pattern <input> Вказує регулярний вираз, яке значення елементу <input> перевіряється
placeholder <input>, <textarea> Вказує коротку підказку, яка описує очікуване значення елемента
poster <video> Визначає зображення, яке буде показано під час завантаження відео, або доки користувач не натисне кнопку відтворення
preload <audio>, <video> Вказує, якщо автор вважає, що аудіо/відео потрібно завантажувати під час завантаження сторінки
readonly <input>, <textarea> Вказує, що елемент доступний лише для читання
rel <a>, <area>, <link> Вказує зв'язок між поточним документом і пов'язаним документом
required <input>, <select>, <textarea> Вказує, що елемент необхідно заповнити перед поданням форми
reversed <ol> Вказує, що порядок списків має бути низхідним (9,8,7...)
rows <textarea> Визначає видиме число рядків у текстовій області
rowspan <td>, <th> Визначає кількість рядків, яким повинна відповідати чарунка таблиці
sandbox <iframe> Вмикає додатковий набір обмежень для вмісту в <iframe>
scope <th> Вказує, чи є чарунка заголовка заголовком для стовпця, рядка або групи стовпців чи рядків
selected <option> Вказує, що параметр повинен бути попередньо вибраний під час завантаження сторінки
shape <area> Визначає форму області
size <input>, <select> Визначає ширину символів (для <input>) або вказує кількість видимих опцій (для <select>)
sizes <img>, <link>, <source> Визначає розмір пов'язаного ресурсу
span <col>, <colgroup> Визначає кількість стовпців, які потрібно охопити
spellcheck Глобальні атрибути Вказує, чи має елемент перевіряти правопис і граматику чи ні
src <audio>, <embed>, <iframe>, <img>, <input>, <script>, <source>, <track>, <video> Вказує URL-адресу мультимедійного файлу
srcdoc <iframe> Визначає вміст HTML сторінки, яка буде відображатися в <iframe>
srclang <track> Визначає мову текстових даних доріжки (обов'язково, якщо kind="subtitles")
srcset <img>, <source> Визначає URL-адресу зображення для використання в різних ситуаціях
start <ol> Вказує початкове значення впорядкованого списку
step <input> Визначає допустимі інтервали чисел для поля вводу
