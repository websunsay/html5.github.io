Визначення та використання

Оголошення <!DOCTYPE> має бути найпершою справою у вашому HTML документі, перед <html> тегом.

Оголошення <!DOCTYPE> не є HTML тегом; це інструкція для веб-браузера, що повідомляє, в якій версії HTML написана сторінка.

В HTML 4.01, оголошення <!DOCTYPE> відноситься до DTD, тому що HTML 4.01 базувався на SGML. DTD визначає правила для мови розмітки, так щоб браузери візуалізували вміст правильно.

HTML5 не заснований на SGML, і тому не вимагає посилання на DTD.

Порада: Завжди додавайте оголошення <!DOCTYPE> до HTML-документів, щоб браузер знав, який тип документа слід очікувати.

Загальні оголошення DOCTYPE

HTML 5

<!DOCTYPE html>

HTML 4.01 Strict

Цей DTD містить всі елементи та атрибути HTML, але НЕ ВКЛЮЧАЄ презентаційні або застарілі елементи (наприклад, шрифт). Фрейми не дозволені.

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">

HTML 4.01 Transitional

Цей DTD містить усі HTML-елементи та атрибути, включаючи презентаційні та застарілі елементи (наприклад, шрифт). Фрейми не дозволені.

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">

HTML 4.01 Frameset

Цей DTD дорівнює HTML 4.01 Transitional, але дозволяє використовувати вміст фреймів.

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Frameset//EN" "http://www.w3.org/TR/html4/frameset.dtd">

XHTML 1.0 Strict

Цей DTD містить всі елементи та атрибути HTML, але НЕ ВКЛЮЧАЄ презентаційні або застарілі елементи (наприклад, шрифт). Фрейми не дозволені. Розмітка також повинна бути записана як добре сформований XML.

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">

XHTML 1.0 Transitional

Цей DTD містить усі HTML-елементи та атрибути, включаючи презентаційні та застарілі елементи (наприклад, шрифт). Фрейми не дозволені. Розмітка також повинна бути записана як добре сформований XML.

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">

XHTML 1.0 Frameset

Цей DTD дорівнює XHTML 1.0 Transitional, але дозволяє використовувати вміст фреймів.

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Frameset//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-frameset.dtd">

XHTML 1.1

Цей DTD дорівнює XHTML 1.0 Strict, але дозволяє додавати модулі (наприклад, для забезпечення підтримки Ruby для східноазіатських мов).

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.1//EN" "http://www.w3.org/TR/xhtml11/DTD/xhtml11.dtd">
