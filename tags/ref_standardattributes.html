HTML атрибути надають елементам сенс і контекст.

Глобальні атрибути нижче можна використовувати на будь-якому HTML елементі.

HTML Глобальні атрибути

= Атрибут доданий в HTML5.

Атрибут Опис
accesskey Визначає клавішу швидкого доступу для активації/фокусування елемента
class Визначає одне або кілька імен класів для елемента (відноситься до класу в таблиці стилів)
contenteditable Вказує, чи можна редагувати вміст елемента
data-* Використовується для зберігання приватних даних на сторінці або програмі
dir Вказує напрямок тексту для вмісту елемента
draggable Вказує, чи можна перетягувати елемент
dropzone Визначає, чи будуть перетягувані дані копіюватись, переміщуватись або зв'язуватись, коли вони будуть скинуті
hidden Вказує, що елемент ще не є або вже не є відповідним
id Вказує унікальний ідентифікатор елемента
lang Вказує мову вмісту елемента
spellcheck Вказує, чи має елемент перевіряти правопис і граматику чи ні
style Визначає вбудований стиль CSS для елемента
tabindex Визначає порядок табуляції елемента
title Визначає додаткову інформацію про елемент
translate Вказує, чи слід перекладати вміст елемента

Не викладайте свій код безпосередньо в коментарях, він відображається некоректно. Скористайтесь такими сервісами, як jsfiddle.net, codepen.io, liveweave.com, jsbin.com, збережіть код на будь-якому з цих сервісів і в коментарях дайте на нього посилання. Так буде видно і код, і результат.
