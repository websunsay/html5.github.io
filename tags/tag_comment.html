Визначення та використання

Тег коментаря використовується для вставки коментарів у вихідний код. Коментарі не відображаються в браузерах.

Ви можете використовувати коментарі, щоб пояснити свій код, який може допомогти вам у подальшому редагуванні вихідного коду. Це особливо корисно, якщо у вас багато коду.

Поради та примітки

Ви також можете використовувати тег коментарів, щоб "приховати" скрипти від браузерів без підтримки сценаріїв (щоб вони не показували їх як звичайний текст):

<script type="text/javascript">
<!--
function displayMsg() {
  alert("Hello World!")
}
//-->
</script> 

Примітка: Дві передніх риски в кінці рядка коментарів (//) є символом коментаря JavaScript. Це запобігає виконанню JavaScript --> тега.

Не викладайте свій код безпосередньо в коментарях, він відображається некоректно. Скористайтесь такими сервісами, як jsfiddle.net, codepen.io, liveweave.com, jsbin.com, збережіть код на будь-якому з цих сервісів і в коментарях дайте на нього посилання. Так буде видно і код, і результат.
